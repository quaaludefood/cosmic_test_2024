### To run the code

Root directory contains the input_file defined in main.py. Navigate to the root directory and run:

```
cd cosmic_test_2024
python main.py
```

The output will be prnited in the terminal.

### To run the unit tests

Navigate to the root directory and run:

```
cd cosmic_test_2024
python tests.py
```
